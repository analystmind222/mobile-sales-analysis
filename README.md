# Mobile Sales Dashboard (Power BI)

## Objective
Analyze mobile sales data to identify trends and business insights.

---

## Tools Used
- Power BI | Dashboard & Visualization 
- Power Query | Data Cleaning & Transformation 
- DAX | Calculated Measures & KPIs 

---

## Data Preparation
- Cleaned and transformed raw data using Power Query
- Removed duplicates and handled missing values
- Created calculated columns

---

## Dashboard Features
- Sales by City & Brand – Geographic and brand-level performance
- Monthly Trends – Time-series analysis of sales volume
- Payment Method Analysis – UPI, Cash, Card breakdown
- Customer Ratings – Satisfaction distribution across products

---

## Key Insights
- Top 3 cities contributed ~40% revenue
- Seasonal sales trends identified
- Ratings mostly above 4
- UPI was dominant payment method
